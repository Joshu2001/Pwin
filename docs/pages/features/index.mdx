# Platform Features

Core features of the Regaarder platform for creators and fans.

## Topics

- **Boosts System** - Premium content promotion
- **Metrics & Analytics** - User and creator analytics
- **Video Overlays** - Interactive video overlays
- **Content Requests** - Fan request system
- **First Time User** - Onboarding for new users

## Feature Overview

Regaarder enables creators to:
- Boost content for increased visibility
- Track performance with detailed metrics
- Add interactive overlays to videos
- Manage fan requests and engagement
- Welcome new users with onboarding flows
